# Football Match Outcome Prediction using Machine Learning

Machine Learning project focused on predicting football match outcomes using only half-time statistics from LaLiga matches.

The objective of the project is to analyze whether events occurring during the first half provide enough information to accurately predict the final result of a football match.

The project combines data analysis, feature engineering, multiclass classification, and model evaluation using several Machine Learning techniques.
